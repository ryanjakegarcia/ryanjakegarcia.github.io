Connect4 Solver

Problem: Find optimal Connect4 moves quickly enough for interactive use.

Approach: Game tree search with pruning, board evaluation heuristics, and efficient state handling.

Result: Produces strong move choices and explains why a move is preferred.

Simple Raycaster

Problem: Render pseudo-3D scenes from a 2D map with minimal complexity.

Approach: Per-column ray marching, distance correction, and texture or color shading by depth.

Result: A compact graphics prototype that demonstrates core 3D rendering principles.
